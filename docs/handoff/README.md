# BioProtect - Paquete de traspaso para sesión nueva

Este ZIP sirve como handoff completo para una sesión nueva de ChatGPT.
Sube este ZIP al chat nuevo y pídele que lea primero estos documentos antes de construir la app.

## Objetivo
Crear una app Android llamada **BioProtect** para un **POCO F6 Pro** con **Xiaomi HyperOS 2.0.210.0.VNKEUXM** y **Android 15**.

## Contenido
- `01_resumen_del_encargo.md`
- `02_especificacion_funcional.md`
- `03_datos_del_dispositivo_y_rutas.md`
- `04_decisiones_cerradas.md`
- `05_prompt_para_nueva_sesion.txt`
- `06_archivos_aportados.md`
- carpeta `referencias_apk/`
- carpeta `capturas/`

## Importante
Las APKs de referencia NO deben reutilizarse como base del proyecto. Solo pueden servir para estudio técnico secundario.
La app debe construirse desde cero.
